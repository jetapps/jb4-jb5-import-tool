# Migrating Configurations from JetBackup 4 to JetBackup 5

This script will allow you to transfer over any supported configurations on JetBackup 4 over to your new installation of JetBackup 5.

# Requirements
1. Have JetBackup 5 installed alongside JetBackup 4.
2. The JetBackup 5 must be a fresh installation.
3. Run the script at the top level of this directory `sh jb4tojb5Wizard.sh`
